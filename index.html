Welcome to MorpherJS v2.0

MorpherJS is a powerful JavaScript library for creating smooth morphing effects between multiple images using HTML5 Canvas and mesh deformation techniques.

New in v2.0: Complete rewrite in modern ES6+ JavaScript, Vite build system, multiple module formats (ESM, CJS, UMD), and significant performance improvements.
